# QQ音乐资源下载扩展

Chrome 浏览器扩展，用于自动捕获 QQ 音乐网页版的音视频资源，支持自动遍历播放列表、批量下载打包。

## ✨ 功能特性

- 🎵 **资源自动捕获**：自动监控网页加载的音视频资源，支持 mp4/m4a/mp3 等主流音视频格式，自动过滤非音视频文件
- ⏯️ **自动顺序播放**：自动遍历 `.songlist__list` 歌曲列表，可自定义切歌间隔，支持开始/停止控制
- 🧹 **智能去重**：按歌曲名自动去重，相同歌曲仅保留最新记录
- 📤 **数据导出**：支持导出歌曲列表为 JSON 格式，包含歌名、下载地址等信息
- 📦 **批量打包下载**：一键批量下载所有捕获的音视频资源，自动打包为 ZIP 文件，文件名自动清理特殊字符，解决 macOS 解压乱码问题
- 🔄 **实时状态同步**：实时显示当前播放歌曲、播放进度、捕获资源数量

## 🚀 安装方法

1. 打开 Chrome 浏览器，进入扩展管理页面：`chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击左上角「加载已解压的扩展程序」
4. 选择本项目根目录，扩展即安装完成

## 📖 使用说明

1. 打开 QQ 音乐网页版，进入你要下载的歌单页面
2. 点击浏览器工具栏中的扩展图标打开弹窗
3. 点击「▶️ 开始自动播放」按钮，扩展会自动顺序播放列表中的所有歌曲
4. 播放过程中，捕获到的音视频资源会实时显示在下载列表中
5. 播放完成后，点击「📥 批量下载」按钮，所有歌曲会自动打包为 ZIP 文件下载到本地
6. 可选操作：
   - 点击「📤 导出JSON」导出所有歌曲信息
   - 点击「🗑️ 清空记录」清空当前捕获的所有资源
   - 点击单条记录的「复制链接」复制资源下载地址
   - 点击单条记录的「重新下载」单独下载某首歌曲

## ⚙️ 自定义配置（可选）

如果需要调整默认参数，可以修改以下文件：

- **切歌间隔**：修改 `content/content.js` 第 174 行的定时器数值，单位毫秒（默认 2000 即 2 秒）
- **列表刷新间隔**：修改 `popup/popup.js` 第 22 行的定时器数值，单位毫秒（默认 1000 即 1 秒）
- **音视频格式支持**：修改 `background/background.js` 第 2-3 行的 `mediaExtensions` 数组，添加你需要支持的格式

## ⚠️ 注意事项

1. 使用前请确保你已经登录 QQ 音乐网页版，有权限收听歌单中的歌曲
2. 播放过程中请保持 QQ 音乐页面处于前台状态，不要最小化浏览器，否则可能导致资源捕获失败
3. 非 VIP 账号只能下载标准音质的歌曲，VIP 账号可下载对应音质的资源
4. 本扩展仅供个人学习使用，请勿用于商业用途，下载的资源版权归腾讯音乐所有
5. 如果遇到资源捕获失败，可以刷新歌单页面后重新尝试

