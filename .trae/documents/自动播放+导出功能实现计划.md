# QQ音乐自动播放+导出功能 - 实现计划

## 功能概述
### 新增1：自动顺序播放功能
点击「开始」按钮后，自动顺序点击`.songlist__list`列表中的歌曲，每5秒点击下一首，直到最后一首自动停止。点击「停止」可中断播放。

### 新增2：按歌曲名去重
所有捕获的媒体资源按歌曲名去重，相同歌曲名只保留最新的一条记录。

### 新增3：导出功能
一键导出所有歌曲信息和对应的URL，支持JSON/CSV格式。

---

## 核心功能点
### 一、自动播放功能
1. **界面修改**
   - 原有「刷新」按钮改为状态切换按钮：
     - 未运行：「▶️ 开始自动播放」（绿色）
     - 运行中：「⏸️ 停止自动播放」（红色）
   - 显示播放进度：`正在播放：歌曲名 (第N首/共M首)`

2. **播放逻辑**
   - 读取页面`.songlist__list`下的所有`<li>`歌曲项
   - 顺序点击每首歌的`.list_menu__play`播放按钮
   - 每5秒点击下一首，播放到最后自动停止
   - 支持手动停止，重置状态

### 二、按歌曲名去重
1. **去重规则**
   - 唯一标识：歌曲名（不区分大小写和特殊字符）
   - 相同歌曲名只保留最新捕获的一条记录
   - 新捕获的同歌名资源覆盖旧记录

2. **去重时机**
   - 捕获到新媒体资源时自动去重
   - 列表展示前自动去重

### 三、导出功能
1. **导出内容**
   - 歌曲名、歌手
   - 下载URL、文件大小
   - MIME类型、捕获时间
2. **导出格式**
   - 默认导出JSON格式，包含完整信息
   - 可选导出CSV格式，方便Excel打开
3. **导出文件命名**
   - `QQ音乐歌曲列表_YYYYMMDD_HHMM.json`
   - `QQ音乐歌曲列表_YYYYMMDD_HHMM.csv`

## 技术实现要点
### 去重逻辑
```javascript
function isDuplicate(songName) {
  for (const [, resource] of mediaResources) {
    if (resource.songName === songName) {
      return true; // 相同歌名视为重复
    }
  }
  return false;
}
```

### 导出CSV格式
- 表头：歌曲名,歌手,URL,大小(MB),类型,捕获时间
- 特殊字符自动转义，避免CSV格式错误

### 播放控制
- 使用`setInterval`实现5秒间隔
- 停止或完成时立即清理定时器
- 页面unload时自动停止播放，避免异常
